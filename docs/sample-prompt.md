## Домен: платформа онлайн-курсов

Создай приложение в `/tmp/course-platform`.

Ресурсы:
- **Organization** — школа, тариф (free/pro/enterprise)
- **Instructor** — работает в организации, ведёт курсы
- **Course** — принадлежит организации и инструктору, цена, лимит студентов, статус (draft → published → archived). Нельзя опубликовать без хотя бы одного урока.
- **Student** — регистрируется в организации
- **Enrollment** — студент записывается на курс. Нельзя записаться дважды. Нельзя на неопубликованный. Если курс полный — ошибка.
- **Lesson** — принадлежит курсу, позиция (порядок), длительность
- **Review** — студент оставляет отзыв на курс, рейтинг 1-5, только если записан
- **Payment** — привязана к записи, статус (pending → confirmed → refunded)

Роли: admin (всё), instructor (свои курсы/уроки), student (запись, отзывы), guest (только чтение опубликованных).

Пройди весь путь: спеки → IR → генерация → бизнес-логика.
